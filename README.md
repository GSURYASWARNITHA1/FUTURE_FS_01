## 🚀 Deploying to GitHub Pages

1. Push all files to a GitHub repository (e.g. `GSURYASWARNITHA1.github.io`)
2. Go to **Settings → Pages**
3. Set source to **main branch / root**
4. Your site will be live at `https://GSURYASWARNITHA1.github.io`
